# QQ 机器人 API 知识库

## 消息收发

| 文件 | 说明 |
|------|------|
| [qq-send.txt](qq-send.txt) | 发送消息 - 单聊/群聊/子频道/私信 |
| [qq-rich-media.txt](qq-rich-media.txt) | 富媒体消息 - 图片/视频/语音/文件上传 |
| [qq-reset.txt](qq-reset.txt) | 撤回消息 |
| [qq-event.txt](qq-event.txt) | 消息事件 |
